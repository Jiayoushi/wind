This package contains the classes that represent the commands that the NanoSQL database recognizes. All of the commands are represented as subclasses of the {@link Command} abstract base-class. Commands are grouped into the following categories:

Data Definition Language (DDL) Commands

These commands manipulate the contents of the data-dictionary. The supported commands are as follows:
  • CREATE TABLE - {@link CreateTableCommand} class
  • DROP TABLE - {@link DropTableCommand} class

Data Manipulation Language (DML) Commands

These commands retrieve and modify the database tables themselves. Note that these commands all derive from the {@link QueryCommand} abstract class. The supported commands are as follows:
  • DELETE ... - {@link DeleteCommand} class
  • INSERT ... - {@link InsertCommand} class
  • SELECT ... - {@link SelectCommand} class
  • UPDATE ... - {@link UpdateCommand} class

Transaction-Demarcation Commands

These commands provide control over when a transaction is started or ended:
  • BEGIN [WORK] | START TRANSACTION - {@link BeginTransactionCommand} class
  • COMMIT [WORK] - {@link CommitTransactionCommand} class
  • ROLLBACK [WORK] - {@link RollbackTransactionCommand} class

Utility Commands

These commands perform various utility operations:
  • ANALYZE ... - {@link AnalyzeCommand} class
  • EXPLAIN ... - {@link ExplainCommand} class
  • EXIT | QUIT - {@link ExitCommand} class
